# E-Commerce Website (Practice Project)

This project is a **practice-based e-commerce website** built as part of a **learning and training process**.

The implementation was done by **following a tutorial step by step**, with the main goal of:
- Practicing real-world frontend development
- Understanding how different parts of an e-commerce system work together
- Improving skills in HTML, CSS, JavaScript, responsiveness, and basic integrations

>  **Disclaimer**  
> This project is **not an original product idea** and was **not designed from scratch independently**.  
> It was developed by applying and extending concepts demonstrated in a tutorial for educational purposes.

---

##  Features Implemented

- Responsive layout for all screen sizes
- Product sliders and category sections
- Shopping cart with full functionality:
  - Add / remove items
  - Quantity control (+ / -)
  - Dynamic total price calculation
  - Cart persistence after page refresh
- Checkout page with:
  - Order summary (dynamic)
  - Delivery address section
  - Coupon code section (UI)
- Checkout submission logic:
  - Order data sent to Google Sheets using Google Apps Script
- Clean and structured Git commit history following best practices

---

##  Technologies Used

- HTML5
- CSS3 (Responsive Design & Media Queries)
- JavaScript (Vanilla JS)
- Google Apps Script (as a lightweight backend)
- Git & GitHub

---

##  Learning Objectives

This project helped me to:

- Apply frontend concepts in a real project structure
- Understand shopping cart and checkout logic
- Work with JavaScript state and DOM manipulation
- Handle responsive UI across multiple breakpoints
- Integrate frontend code with an external service (Google Sheets)
- Practice writing clean, professional Git commits
